<p style="margin-left: 20px;"><strong>1. Pan-Tilt Mechanism :-</strong></p><p style="margin-left: 20px;"><strong>    T</strong>his mechanism uses two servos to give 360 degrees of rotational in space i.e. 2 rotational Degrees of freedom. It can be shown as :</p><p><img src="http://s3.postimg.org/ilrrnieo3/sensor_pan_tilt_manual.jpg" style="width: 431px;"></p><p style="margin-left: 20px;"></p><p style="margin-left: 20px;"><strong>2. Sever Setup :</strong></p><p style="margin-left: 20px;">To setup server to stream motion JPEG we incorporated <a href="https://docs.python.org/2/library/basehttpserver.html">BaseHTTPServer</a> library of python which uses BaseHTTPRequestHandler and HTTPServer . Server once started and connected to a Wireless Network with or without internet access responses to requests that are handled by BaseHTTPRequestHandler at Port  8080. Server encode images into motion JPEG images for Wireless transmission with is then decoded by android application.</p><p style="margin-left: 20px;"></p><p style="margin-left: 20px;"><strong>3. Image processing : </strong></p><p style="margin-left: 20px;"><strong></strong> We have done image processing using <a href="http://opencv-python-tutroals.readthedocs.org/en/latest/py_tutorials/py_tutorials.html">OpenCV library of python</a> . First of all image is converted from RBG to HSV color space. Blue color is extracted from image by thresholding and smoothing of image to create a mask. Contour is detected on mask created and using moments we found out center of the ball.</p><p style="margin-left: 20px;"></p><p style="margin-left: 20px;"><strong>4.   PID algorithm :</strong></p><p style="margin-left: 20px;"> To make webcam follow the ball we implemented <a href="http://brettbeauregard.com/blog/2011/04/improving-the-beginner%E2%80%99s-pid-sample-time/">PID</a> on our micro-controller which uses distance moved by center of ball as its error. PID algorithm is applied on this  error to to  find out servos speed and angle. In real time when object moves in front of webcam the angle and the speed of both servos used in pan-tilt have different angular velocities.</p><p style="margin-left: 20px;"></p><p style="margin-left: 20px;"><strong>5. Android Application :</strong>   </p><p style="margin-left: 20px;"> IDE used :-<a href="http://developer.android.com/tools/studio/index.html"> Android studio</a></p><p style="margin-left: 20px;">Features :-</p><p style="margin-left: 40px;">1. Basic buttons objects</p><p style="margin-left: 40px;">2. Event listeners and call back methods on the buttons were reated to change activities or to connect to the Bluetooth.</p><p style="margin-left: 40px;">3. Bluetooth part of android app :</p><p style="margin-left: 60px;"> (i) Contains buttons to connect to already paired  Bluetooth device. (i.e to the HC-05 bluetooth module on our micro-controller)</p><p style="margin-left: 60px;">(ii) Sends required data in the form of strings to the connected Bluetooth module when a particular button (in the app) is pressed.</p><p style="margin-left: 60px;">(iii) Closes the Bluetooth connection when an activity gets destroyed.</p><p style="margin-left: 40px;">4. Video Streaming: - Streaming motion JPEG images (i.e. we can see what camera can see) in the android application by accessing and listening the port of created</p><p style="margin-left: 180px;">     wireless network.</p><p style="margin-left: 20px;">  </p>
