<p><strong><span style="color: rgb(31, 73, 125);">The aim of our project is to make a biped robot.The robot can perform actions such as moving forward,moving backward,turning left,turning right,moving arms to lift an object etc.It can be controlled wirelessly through a laptop using bluetooth.</span></strong></p><p><strong><span style="color: rgb(31, 73, 125);">The bot will have an ultrasound sensor which will serve as eyes,and measure distance of any obstacle in front of it.The Interface between the robot and laptop will be a GUI created in a software named Processing.The user can give commands to the bot through this interface.</span></strong></p><p><strong><span style="color: rgb(31, 73, 125);">There wiil be two modes of the bot-manual and auto.In manual mode the user controls the action of the bot,while in auto mode the bot moves on itself detecting and avoiding obstacles.</span></strong></p>
