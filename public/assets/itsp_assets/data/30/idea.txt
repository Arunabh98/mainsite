<p>Components of Remote:</p><ol><li>Accelerometer</li><li>Arduino UNO</li><li>R433A RF Transmitter</li><li>9V Battery</li></ol><p>Components of Bot:</p><ol><li>Arduino MEGA</li><li>R433A RF Receiver</li><li>TIP122 Transistor circuit</li><li>DPDT Relay switches</li><li>Ultrasonic Sensors</li><li>Buzzer</li></ol><p>Working:</p><p style="margin-left: 40px;">According to tilt of accelerometer (components of gravitational acceleration<em> g</em>), it provides voltage readings to arduino UNO. Integer values cannot be sent by RF module R433A. Integers readings are converted to char array by itoa function. VirtualWire library is used to communicate between two aduinii using R433A RF Module.</p><p style="margin-left: 40px;">Receiver at Arduino MEGA receives char readings and using atoi function converts these to original integer values. These integer values are used to control the speed and direction of the bot.</p><p style="margin-left: 40px;">Ultrasonic sensors plugged in arduino MEGA returns distance of nearest object in cm. Using these values, we ensure that the bot does not collide with any obstacle.</p><p style="margin-left: 40px;">       </p>
